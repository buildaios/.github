# .github

This repository contains the organization-wide community health files and default GitHub configuration for **BuildAIOS**.

These files are automatically applied to repositories across the BuildAIOS organization unless a repository provides its own overrides.

## Purpose

This repository helps maintain a consistent contributor experience across all BuildAIOS projects by providing shared:

- Contribution guidelines
- Code of Conduct
- Security policy
- Pull Request template
- Issue templates
- Organization profile README

## Repository Structure

```text
.
├── profile/
│   └── README.md                 # Organization landing page
│
├── CONTRIBUTING.md               # Default contribution guide
├── CODE_OF_CONDUCT.md            # Community standards
├── SECURITY.md                   # Vulnerability reporting policy
├── PULL_REQUEST_TEMPLATE.md      # Default pull request template
│
└── ISSUE_TEMPLATE/
    ├── bug_report.yml
    ├── feature_request.yml
    ├── documentation.yml
    └── config.yml
```

## Organization

BuildAIOS is an open-source initiative focused on building a governed AI runtime and cognitive control plane on Fedora Linux.

The project develops infrastructure for:

- Governed AI agents
- Runtime orchestration
- Model Context Protocol (MCP)
- Trust-aware execution
- Workflow automation
- Environment Packs
- AI security and governance

## Main Repositories

| Repository | Purpose |
|------------|---------|
| Runtime-agent | Agent runtime and orchestration |
| MCP-gateway | Secure MCP routing and policy enforcement |
| Docs | Architecture, specifications and documentation |
| aios-website | Official website and public documentation |
| .github | Shared GitHub community configuration |
